Module Unmark.Attr

module Attr: sig .. end

Benchmark metadata.

type t

Attributes are dictionaries that can be attached to benchmarks.

val empty : t

Empty set of attributes.

val is_empty : t -> bool

is_empty t is true iff t is empty.

val key : 'a Unmark.fmt -> name:string -> 'a -> t

key pp ~name v is a set of attributes that contains only the mapping name -> v.

val (++) : t -> t -> t

t1 ++ t2 is the right-biased union of t1 and t2.

val pp : t Unmark.fmt

pp ppf t pretty-prints t on ppf.
